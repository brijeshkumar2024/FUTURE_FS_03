# Spice Garden Website Demo

A responsive restaurant website for the local business **Spice Garden**, built with plain HTML, CSS, and JavaScript.

## What is included

- Premium dark luxury restaurant design with glassmorphism accents
- Animated cinematic hero section with parallax background and review badge
- Sticky glass navbar with smooth scrolling links
- About Us section with chef profile and animated experience counters
- Menu section with elegant category pills and hover-driven cards
- Masonry gallery with premium zoom lightbox experience
- Modern testimonial cards with carousel navigation
- Luxury reservation booking panel with polished form fields
- Embedded Google Map for location
- Floating WhatsApp CTA with contemporary button styling
- Scroll progress indicator, loading animation, and immersive motion
- Fully responsive layout optimized for mobile and desktop

## How this helps the business

- Improves local discoverability with clear branding, menu details, and reservation options.
- Encourages bookings through a prominent CTA and easy contact flow.
- Builds trust with customer testimonials and chef story.
- Supports mobile traffic with responsive design and quick WhatsApp contact.

## Files

- `index.html` — website structure
- `styles.css` — styling and responsive layout
- `script.js` — interactive tabs, carousel, and lightbox

## Viewing the demo

Open `index.html` in a browser to view the live demo locally, or visit the live deployment:
